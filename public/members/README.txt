Add the four member photos to this folder using these filenames:

- ennis.jpg
- liew.jpg
- chua.jpg
- tai.jpg

Square or portrait photos work best. The carousel crops each image to fit a
portrait card. If you use different filenames or PNG/WebP files, update the
corresponding image paths in src/main.js under indexMemberData.

If an image is missing, the website automatically shows that member's initials.
